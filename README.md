## Getting Started

Folder structure will be like

```
	ac.k
	├── .dockerignore
	├── .env.example
	├── .eslintignore
	├── .eslintrc
	├── .gitignore
	├── .prettierrc 
	├── cspell.json
	├── docker-compose.yml 
	├── dockerfile 
	├── LICENSE.md
	├── nest-cli.json
	├── package.json
	├── README.md
	├── tsconfig.build.json
	├── tsconfig.build
	└── src
		├── app
		├── auth
		├── config
		├── database
		├── helper
		├── language
		├── middleware
		├── pipe
		├── response
		├── user
		└── main.ts
		 
```
